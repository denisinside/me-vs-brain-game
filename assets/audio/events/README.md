# Event Audio Files

Помістіть аудіо файли подій у цю директорію.

## Необхідні файли:

### Подія 1: Стукіт у двері
- `event1_1.mp3` - Звук після "Відчинити двері"
- `event1_2.mp3` - Звук після "Працювати далі"

### Подія 2: Дзвінок батька
- `event2_1.mp3` - Звук після "Пожартувати"
- `event2_2.mp3` - Звук після "Накричати"

### Подія 3: Розрядився ноут!
- `event3_success.mp3` - Звук успіху
- `event3_failure.mp3` - Звук провалу

### Подія 4: Геніальна ідея!
- `event4_1.mp3` - Звук після "Записати ідею"
- `event4_2.mp3` - Звук після "Ігнорувати"

### Подія 5: Друг просить допомоги
- `event5_1.mp3` - Звук після "Відповісти"
- `event5_2.mp3` - Звук після "Ігнорувати"

### Подія 6: Хочеш мандаринку?
- `event6_1.mp3` - Звук після "Ловити мандаринку"
- `event6_2.mp3` - Звук після "Стримувати бажання"

### Подія 7: Спокуса з Гітхабу
- `event7_1.mp3` - Звук після "Надихнутися"
- `event7_2.mp3` - Звук після "Дослідити"

### Подія 8: Екстрена тривога!
- `event8_success.mp3` - Звук успіху
- `event8_failure.mp3` - Звук провалу

### Подія 9: (не)Планове відключення світла
- `event9_1.mp3` - Звук після "Роздати мобільний інтернет"
- `event9_2.mp3` - Звук після "Включити генератор"

## Формат:
- Тип: MP3
- Якість: 128-192 kbps
- Тривалість: 2-5 секунд

## Примітки:
- Звуки програються через 500мс після завершення відео результату
- Можна використати ElevenLabs для генерації реплік студента
- Або просто звукові ефекти (смішні звуки, музика, тощо)
